# ADR 索引

本目录存放 dashboard 的 Architecture Decision Records（Nygard 格式）。每个决策一个文件，文件名 `NNNN-{title}.md`，序号连续。

## ADR 列表

尚无 ADR。新增请用 `pnpm new-adr --title="..."`（工具上线前可手工创建）。

## Nygard 模板

每个 ADR 必须包含：

- `## Status` (proposed / accepted / deprecated / superseded)
- `## Context`
- `## Decision`
- `## Consequences`
